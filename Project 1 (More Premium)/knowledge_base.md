## Grading Scale (NSU Standard)

| Letter | Grade Point | Letter | Grade Point |
|--------|------------|--------|------------|
| A      | 4.00       | C      | 2.00       |
| A-     | 3.70       | C-     | 1.70       |
| B+     | 3.30       | D+     | 1.30       |
| B      | 3.00       | D      | 1.00       |
| B-     | 2.70       | F      | 0.00       |
| C+     | 2.30       |        |            |

### Non-GPA Entries
- **W** — Withdrawn (not counted in credits or GPA)
- **I** — Incomplete (not counted in credits or GPA)
- **P** — Pass (credits counted, not in GPA)
- **TR** — Transfer credit (credits counted, not in GPA)

### Rules
- Minimum CGPA for graduation: **2.00**
- CGPA below 2.00 triggers **Academic Probation**
- **F** grade: 0 credits earned, 0.00 included in GPA calculation
- **0-credit courses**: Not counted toward total earned credits, but must be completed; included in GPA only if credit > 0
- **Retakes**: Only the **best grade** is used for CGPA calculation (per NSU official policy)
