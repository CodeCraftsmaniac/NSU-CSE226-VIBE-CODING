## Grading System

| Grade | Points | Description |
|-------|--------|-------------|
| A     | 4.00   | Excellent   |
| A-    | 3.70   | Very Good   |
| B+    | 3.30   | Good Plus   |
| B     | 3.00   | Good        |
| B-    | 2.70   | Fair Plus   |
| C+    | 2.30   | Fair        |
| C     | 2.00   | Average     |
| C-    | 1.70   | Below Avg   |
| D+    | 1.30   | Poor Plus   |
| D     | 1.00   | Poor        |
| F     | 0.00   | Fail        |

### Special Grades
- W: Withdrawn (no credit, no GPA impact)
- I: Incomplete (must be resolved)
- P: Pass (credit earned, no GPA impact)
- TR: Transfer (credit earned, no GPA impact)

## Tuition Waiver Requirements

| CGPA Range | Waiver % | Scholarship Name |
|------------|----------|------------------|
| >= 3.97    | 100%     | Chancellor's Award |
| >= 3.75    | 50%      | Vice-Chancellor's Scholarship |
| >= 3.50    | 25%      | Dean's Scholarship |

*Note: Requires minimum 30 completed credits for waiver eligibility*
