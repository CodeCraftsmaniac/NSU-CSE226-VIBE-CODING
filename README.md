This is the official project repository for **CSE226 &mdash; Vibe Coding** at **North South University**, Dhaka, Bangladesh &mdash; **Spring 2026**.

**CSE226** is a hands-on, project-driven course teaching students to build production-quality full-stack applications using modern AI-assisted development workflows. The course is built around *vibe coding* &mdash; a paradigm where developers leverage AI copilots, large language models, and intelligent IDEs to architect, implement, debug, and ship software at unprecedented speed and quality.

Students work with real tools used in industry: **Cursor**, **Windsurf**, **GitHub Copilot**, **Gemini CLI**, **Claude**, and **ChatGPT** &mdash; and are assessed through live projects, a midterm, final, and a competitive hackathon.
